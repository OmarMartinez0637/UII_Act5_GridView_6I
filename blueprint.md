# Actividad 5: GridView con Diseño Personalizado

## Descripción General

Esta aplicación de Flutter demuestra el uso de `GridView` para mostrar una colección de imágenes con un diseño de tarjeta personalizado. La aplicación presenta un `AppBar` con un título y un cuerpo que contiene un `GridView`. Cada elemento del `GridView` es una tarjeta que muestra una imagen y un título. El diseño es responsivo y se adapta a diferentes tamaños de pantalla.

## Estilo y Diseño

La aplicación utiliza los siguientes elementos de estilo y diseño:

*   **Paleta de colores:** La aplicación utiliza una paleta de colores simple con un `AppBar` de color azul y un fondo blanco.
*   **Tipografía:** La aplicación utiliza la fuente predeterminada de Flutter.
*   **Iconografía:** La aplicación no utiliza iconos.
*   **Imágenes:** La aplicación utiliza imágenes de Platzhalter de la red.
*   **Componentes:**
    *   `AppBar`: Muestra el título de la aplicación.
    *   `GridView`: Muestra la colección de imágenes.
    *   `Card`: Contiene cada imagen y título.
    *   `Image`: Muestra la imagen.
    *   `Text`: Muestra el título.

## Características

*   Muestra una colección de imágenes en un `GridView`.
*   Utiliza un diseño de tarjeta personalizado para cada elemento del `GridView`.
*   Es responsivo y se adapta a diferentes tamaños de pantalla.
